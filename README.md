# turin2026_lecture4
Lecture 4 of the ICT4D Master on AI for civic data.

## Pipeline Mapping Exercise                                                                          
                                                                                                       
Map an internal data process from your organisation against the 7 steps of the data pipeline.        
                                                                                                     
### Instructions                                                                                      
                                                                                                     
1. Fork this repository                                                                              
2. Edit `pipeline-map.md` with your group's process
3. Open a Pull Request back to this repository with your group name as the PR title                  
                                                                                                       
### The template                                                                                      
                                                                                                       
Fill in `pipeline-map.md` — one section per pipeline step. For each step, describe what happens in your organisation's process today, and whether AI could help.
